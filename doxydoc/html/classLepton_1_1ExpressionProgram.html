Detailed Description

An ExpressionProgram is a linear sequence of Operations for evaluating an expression. The evaluation is done with a stack. The arguments to each Operation are first taken off the stack in order, then it is evaluated and the result is pushed back onto the stack. At the end, the stack contains a single value, which is the value of the expression.

An ExpressionProgram is created by calling createProgram() on a ParsedExpression.
